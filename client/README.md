# Medical Clinic Frontend (React + Vite)

This frontend is built using React with Vite for fast development and optimized builds.

## Available Scripts

In the client directory, run:

### `npm install`
Installs dependencies

### `npm run dev`
Runs the app in development mode  
Open http://localhost:5173

### `npm run build`
Builds the app for production into the `dist` folder

### `npm run preview`
Preview production build locally

## Notes

- This frontend connects to the Express backend via API routes:
  - `/api/employee`
  - `/api/doctor`
- Make sure the backend server is running on port 3000
